# LegoLens v3.0.1 Runtime

Compact runtime-only branch for LegoLens v3.0.1.

This branch is intentionally separate from `main` and contains only the lightweight runtime files needed to start and evaluate the 3.0.1 Command Center interface.

## Start

```bash
npm install
npm start
```

Open `http://localhost:8787`.

## Included

- Command Center style interface
- New LegoLens logo assets
- Dashboard
- Case Dashboard
- GEO observations
- Review Queue
- Reports and Settings placeholders
- Language switch with RTL handling for Arabic, Farsi and Urdu

No docs screenshots, runtime logs, historical builds or main-branch artifacts are included.
