# Operating Manual

## Starten

```bash
open index.html
```

of met backend:

```bash
node backend/server.mjs
```

## Dagelijkse workflow

1. Open **Today** voor status en open checks.
2. Ga naar **Monitor** voor trends, confidence en source-family review.
3. Gebruik **Investigate** voor lage confidence en evidence gaps.
4. Gebruik **Review** om candidates te accepteren, af te wijzen of terug te zetten.
5. Gebruik **Report** voor briefingproducten.
6. Gebruik **Admin Hub** voor manifesten, release gate en exports.

## Nieuwe items toevoegen

Nieuwe input moet als candidate binnenkomen. Pas na menselijke review wordt het een content item.

## Rapportage

Exporteer rapportage alleen als de onderliggende items voldoende zijn gevalideerd. Markeer onzekerheden expliciet.
