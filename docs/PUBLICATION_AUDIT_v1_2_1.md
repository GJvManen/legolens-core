# Publication audit — LegoLens Core v1.2.1

## Result

LegoLens Core v1.2.1 is prepared as a publication-clean release.

## Publication-clean rules

The public repository should not contain:

- bundled screenshots or generated social graphics
- ZIP, DOCX, PPTX or PDF artifacts
- SQLite, database or local runtime files
- API keys, credentials or `.env` files
- raw sensitive media captures
- social campaign or pitch artifacts

## Expected release contents

The repository should contain:

- source code
- schemas and tests
- case-pack JSON and documentation
- neutral branding documentation
- update and release metadata
- responsible-use guidance
- multilingual publication explanation

## Validation checklist

- README updated to v1.2.1 positioning.
- `package.json` updated to 1.2.1.
- `data/version.json` updated to v1.2.1.
- `data/update_manifest.json` updated to v1.2.1.
- Multilingual publication explanation added.
- Media policy documented as external media packs only.
- Direct publication disabled.
- Human review required.

## Operational note

Case-pack starter content is intended for triage, workflow testing and analyst review. It is not automatically verified and should not be externally published without review and corroboration.
