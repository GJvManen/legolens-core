# LegoLens Core v1.2.1 — Multilingual publication explanation

## English

LegoLens Core is a neutral, review-first intelligence framework for monitoring complex information environments. It helps teams structure signals, claims, sources, evidence and reports in a traceable workflow. Iran is the first case pack; the framework is designed for many cases.

## 中文（简体）

LegoLens Core 是一个中立的、以人工审核为优先的情报分析框架，用于监测复杂的信息环境。它帮助团队把信号、主张、来源、证据和报告组织到可追溯的工作流中。伊朗是第一个案例包；该框架可扩展到更多案例。
