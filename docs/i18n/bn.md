# LegoLens Core v1.2.1 — বাংলা

LegoLens Core একটি নিরপেক্ষ, review-first intelligence framework, যা জটিল তথ্য পরিবেশ পর্যবেক্ষণের জন্য তৈরি। এটি signal, claim, source, evidence এবং report-কে একটি traceable workflow-তে সংগঠিত করতে সাহায্য করে।

Iran প্রথম case pack। frameworkটি আরও অনেক case-এর জন্য ব্যবহারযোগ্য এবং conflict, crisis ও information-environment monitoring-এ সম্প্রসারণযোগ্য।
