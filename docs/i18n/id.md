# LegoLens Core v1.2.1 — Bahasa Indonesia

LegoLens Core adalah framework intelijen yang netral dan mengutamakan review manusia untuk memantau lingkungan informasi yang kompleks. Framework ini membantu tim menyusun sinyal, klaim, sumber, bukti, dan laporan dalam workflow yang dapat ditelusuri.

Iran adalah case pack pertama. Framework ini dirancang untuk banyak konteks dan dapat diperluas dengan case pack tambahan untuk pemantauan konflik, krisis, dan lingkungan informasi.
