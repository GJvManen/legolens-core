# LegoLens Core v1.2.1 — Português

LegoLens Core é um framework neutro de inteligência, com revisão humana em primeiro lugar, para monitorar ambientes de informação complexos. Ele ajuda equipes a estruturar sinais, alegações, fontes, evidências e relatórios em um fluxo rastreável.

O Irã é o primeiro case pack. O framework foi criado para muitos contextos e pode ser ampliado com novos case packs para monitoramento de conflitos, crises e ambientes informacionais.
