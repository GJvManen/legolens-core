# LegoLens Core v1.2.1 — Nederlands

LegoLens Core is een neutraal, review-first intelligence framework voor het monitoren van complexe informatieomgevingen. Het helpt teams om signalen, claims, bronnen, evidence en rapportages samen te brengen in één traceerbare workflow.

Iran is de eerste case pack. Het framework is gebouwd voor veel meer casussen en kan worden uitgebreid voor conflictmonitoring, crisisanalyse en monitoring van informatieomgevingen.
