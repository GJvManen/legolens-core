# LegoLens Core v1.2.1 — Español

LegoLens Core es un framework neutral de inteligencia, basado en revisión humana, para monitorizar entornos de información complejos. Ayuda a estructurar señales, afirmaciones, fuentes, evidencias e informes en un flujo de trabajo trazable.

Irán es el primer case pack. El framework está diseñado para muchos contextos y puede ampliarse con nuevos case packs para monitoreo de conflictos, crisis y entornos informativos.
