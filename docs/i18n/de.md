# LegoLens Core v1.2.1 — Deutsch

LegoLens Core ist ein neutrales, review-first Intelligence-Framework zur Beobachtung komplexer Informationsumgebungen. Es hilft Teams, Signale, Behauptungen, Quellen, Evidenz und Berichte in einem nachvollziehbaren Workflow zu strukturieren.

Iran ist das erste Case Pack. Das Framework ist für viele Kontexte ausgelegt und kann mit weiteren Case Packs für Konflikt-, Krisen- und Informationsumgebungs-Monitoring erweitert werden.
