# LegoLens Core v1.2.1 — हिन्दी

LegoLens Core एक तटस्थ, review-first intelligence framework है जो जटिल सूचना वातावरणों की निगरानी के लिए बनाया गया है। यह संकेतों, दावों, स्रोतों, evidence और रिपोर्टों को एक traceable workflow में व्यवस्थित करने में मदद करता है।

Iran पहला case pack है। यह framework कई अन्य cases के लिए बनाया गया है और conflict, crisis तथा information-environment monitoring में उपयोग किया जा सकता है।
