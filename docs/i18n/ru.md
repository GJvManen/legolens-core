# LegoLens Core v1.2.1 — Русский

LegoLens Core — это нейтральный intelligence framework с приоритетом человеческой проверки для мониторинга сложных информационных сред. Он помогает структурировать сигналы, утверждения, источники, доказательства и отчеты в прослеживаемом рабочем процессе.

Иран — первый case pack. Framework рассчитан на множество сценариев и может расширяться новыми case packs для мониторинга конфликтов, кризисов и информационных сред.
