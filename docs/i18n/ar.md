# LegoLens Core v1.2.1 — العربية

LegoLens Core هو إطار محايد للاستخبارات والتحليل يعتمد على المراجعة البشرية أولاً لمراقبة بيئات المعلومات المعقدة. يساعد الفرق على تنظيم الإشارات والادعاءات والمصادر والأدلة والتقارير ضمن سير عمل قابل للتتبع.

إيران هي أول حزمة حالة. صُمم الإطار ليكون قابلاً للتوسع إلى حالات أخرى في مراقبة النزاعات والأزمات وبيئات المعلومات.
