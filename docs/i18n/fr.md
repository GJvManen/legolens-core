# LegoLens Core v1.2.1 — Français

LegoLens Core est un framework neutre d'intelligence, fondé sur une approche review-first, pour surveiller des environnements informationnels complexes. Il aide les équipes à organiser signaux, affirmations, sources, preuves et rapports dans un flux de travail traçable.

L'Iran est le premier case pack. Le framework est conçu pour de nombreux contextes et peut être étendu avec d'autres case packs pour le suivi des conflits, des crises et des environnements informationnels.
