# Project overview — LegoLens Core v3.0.0

LegoLens Core v3.0.0 is a local-first workspace with a Node.js backend and a static browser interface.

The current release focuses on:

- local project state;
- source records;
- local review workflows;
- audit logging;
- backup and restore;
- local outputs;
- real browser screenshot capture.

See the repository README for the full v3.0.0 system workflow description.
