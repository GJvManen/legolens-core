# LegoLens Iran Case Pack 1.0

This is the first bundled case pack for **LegoLens Core v1.0**. It demonstrates how a complex information environment can be represented as sources, items, evidence, claims, incidents, previews, graph context and report-ready outputs.

|---|---:|

| Evidence records | 476 |
| Claims | 14 |
| Incidents | 6 |

## Important

This case pack is a demonstration of the framework. The method is reusable for other conflict, crisis and fast-changing information environments. Keep the review-first policy: candidates and AI suggestions must be reviewed by a human before becoming published content.
